# scoop-teru

[![Tests](https://github.com/nicholasglazer/scoop-teru/actions/workflows/ci.yml/badge.svg)](https://github.com/nicholasglazer/scoop-teru/actions/workflows/ci.yml) [![Excavator](https://github.com/nicholasglazer/scoop-teru/actions/workflows/excavator.yml/badge.svg)](https://github.com/nicholasglazer/scoop-teru/actions/workflows/excavator.yml)

[Scoop](https://scoop.sh) bucket for [teru](https://github.com/nicholasglazer/teru) -- AI-first terminal emulator, multiplexer, and tiling manager.

## Install

```powershell
scoop bucket add teru https://github.com/nicholasglazer/scoop-teru
scoop install teru
```

## Update

```powershell
scoop update teru
```

## About teru

One binary. No GPU. 1.4MB. Built-in multiplexer, 8 tiling layouts, AI agent protocol, MCP server.
